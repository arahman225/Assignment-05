What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

How do you select an element from the DOM?
  1. Using getElementsByTagName
  2. Using getElementsByClassName
  3. Using querySelector
  4. Using querySelectorAll
  5. Using getElementById: The getElementById method allows you to select an element by its ID
What is event delegation in the context of the DOM, and why is it useful?

Event delegation attaches a single event listener to a parent element, handling events from child elements via event bubbling, improving performance, especially for dynamic content or large DOM structures.
